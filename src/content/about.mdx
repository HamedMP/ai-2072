## 📚 About This Project

This project, AI-2072, is a digital garden template inspired by the [AI-2027](https://ai-2027.com) project. It combines modern web technologies with classic design principles. 🌐✨

## 🚀 Tech Stack

Built with Next.js 15.3, Tailwind CSS v4, and MDX for content management. The beautiful typography comes from ET Book font, giving this site its distinctive academic feel. 🖋️

You can use shadcn/ui components directly in your MDX content, as well as interactive charts like the one below! This makes your digital garden not just beautiful, but functional and data-rich. 📊✨
